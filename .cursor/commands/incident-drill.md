# Incident Drill

Design or run a scoped incident / failure drill for OrderFlow.

1. State objective, blast radius, success criteria, and rollback/stop conditions up front.
2. Prefer local-first; no paid cloud or destructive actions without explicit approval.
3. Execute only approved steps; record actual commands and observations.
4. Produce a short after-action: timeline, what worked, gaps, and follow-ups. Never fabricate drill outcomes.
