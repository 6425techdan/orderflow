# Reliability Review

Review the change for reliability posture (SLOs, probes, drills, observability).

1. Map affected user journeys and failure modes.
2. Check whether SLOs, probes, alerts, and runbooks still match implementation.
3. Verify simulated scale / drill claims are honest and scoped.
4. Output: gaps, risks, and prioritized fixes. Distinguish facts vs assumptions. Do not invent metrics.
